Nocturnal Wildlife Classification via Point Clouds

This project focused on building an end-to-end LiDAR pipeline for nocturnal wildlife detection and classification. The workflow included segmentation, clustering, tracking, preprocessing, and DGCNN-based classification.

The goal was to create a reliable classification system that could operate on point-cloud data under real-world field conditions. A major focus was balancing classification performance with robustness across noisy and incomplete detections.

Results

The system achieved strong classification accuracy on clean ground-truth crops and solid end-to-end performance once integrated into the full detection and tracking pipeline.

Skills Acquired

  • Point-cloud preprocessing
  • Object clustering and tracking
  • Deep learning workflows
  • Data pipeline design
  • Model evaluation
  • Performance analysis

Tools

  • Python
  • LiDAR data
  • DGCNN
  • NumPy / ML tooling
